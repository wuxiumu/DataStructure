# 数据结构树(Tree)详解
 
树是数据结构中比较重要也是比较难理解的一类存储结构。本章主要主要围绕二叉树，对树的存储以及遍历做详细的介绍，同时还会涉及到有关树的实际应用，例如构建哈弗曼编码等。

由于树存储结构本身比较复杂，需要有耐心地去搞清楚每一节中的每个知识点，在学习时，建议从每节的问题出发，搞清楚文章的解题思路。
